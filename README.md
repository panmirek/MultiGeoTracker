# Demo

www.mgt.specteam.pl

Just select your team (any name) and set yours initials (or any 2-chars name) and press Join.

# Technologies

- Angular 2 (v2.4)
- Firebase (angularfire2)
- Google Maps (angular2-google-maps)

- SSL (required for browser navigation functions)
